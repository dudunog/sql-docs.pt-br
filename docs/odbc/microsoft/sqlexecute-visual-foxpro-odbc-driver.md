# <a name="sqlexecute-visual-foxpro-odbc-driver"></a>SQLExecute (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível do núcleo  
  
 Executa uma declaração SQL preparada (uma declaração já preparada pelo [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)). O driver usa os valores atuais das variáveis de marcador de parâmetro se houver algum parâmetro na declaração.  
  
 Para obter mais informações, consulte [SQLExecute](../../odbc/reference/syntax/sqlexecute-function.md) no *Programador ODBC*.
