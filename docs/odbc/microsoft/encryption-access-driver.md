# <a name="encryption-access-driver"></a>Encryption (Driver do Access)
(Este controle só está disponível quando a Versão 4. *x* formato é selecionado.)  
  
 Com a versão 4.0 do Microsoft Jet Database Engine, a criptografia do banco de dados é permitida ao criar bancos de dados access. Consulte o *Programador do Mecanismo de Banco de Dados a Jato* para obter informações adicionais.
