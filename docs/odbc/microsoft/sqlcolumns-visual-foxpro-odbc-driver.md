# <a name="sqlcolumns-visual-foxpro-odbc-driver"></a>SQLColumns (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado em [referência da API do ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: completo  
  
 Conformidade da API ODBC: nível 1  
  
 Para uma tabela, o cria um conjunto de resultados que é a lista de colunas para a tabela ou tabelas especificadas.  
  
 Para obter mais informações, consulte [SQLColumns](../../odbc/reference/syntax/sqlcolumns-function.md) na *referência do programador de ODBC*.
