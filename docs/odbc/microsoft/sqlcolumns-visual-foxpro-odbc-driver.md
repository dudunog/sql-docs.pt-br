# <a name="sqlcolumns-visual-foxpro-odbc-driver"></a>SQLColumns (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível 1  
  
 Para uma tabela, cria um conjunto de resultados que é a lista de colunas para a tabela ou tabelas especificadas.  
  
 Para obter mais informações, consulte [SQLColumns](../../odbc/reference/syntax/sqlcolumns-function.md) no *Programador ODBC .*
