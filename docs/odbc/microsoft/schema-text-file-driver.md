# <a name="schema-text-file-driver"></a>Schema (Driver de Arquivo de texto)
O esquema inclui informações sobre cada tabela (arquivo de texto) em uma fonte de dados, incluindo o formato da tabela, o número de linhas para digitalizar para determinar os tipos de coluna, se a primeira linha da tabela contém nomes de coluna, se o arquivo-fonte é gravado usando uma página de código OEM ou ANSI e o nome, o tipo de dados e a largura de cada coluna.
