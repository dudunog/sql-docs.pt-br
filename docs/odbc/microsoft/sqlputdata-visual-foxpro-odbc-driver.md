# <a name="sqlputdata-visual-foxpro-odbc-driver"></a>SQLPutData (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível 1  
  
 Permite que um aplicativo envie dados para um parâmetro ou coluna para o motorista na hora da execução da declaração.  
  
 Para obter mais informações, consulte [SQLPutData](../../odbc/reference/syntax/sqlputdata-function.md) no *Programador ODBC*.
