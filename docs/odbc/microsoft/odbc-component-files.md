# <a name="odbc-component-files"></a>Arquivos de componentes ODBC
Os arquivos do Driver ODBC são exigidos por cada componente do ODBC Desktop Database Drivers 4.0.  
  
 Alguns arquivos são exigidos por mais de um componente. Esses arquivos devem ser redistribuídos se você pretende redistribuir qualquer um dos Drivers de banco de dados desktop da ODBC com seus aplicativos comerciais.  
  
 Esses arquivos estão instalados no diretório \Windows\System para Windows 95 ou posterior, ou no diretório \Windows\System32 para Windows NT 3.51 ou posterior e Windows 2000. Se os arquivos ODBC foram previamente instalados em um diretório diferente, certifique-se de usar os arquivos mais novos no diretório \Windows\System (ou System32).
