# <a name="sqlparamdata-visual-foxpro-odbc-driver"></a>SQLParamData (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível 1  
  
 Usado em conjunto com [o SQLPutData](../../odbc/microsoft/sqlputdata-visual-foxpro-odbc-driver.md) para especificar dados de parâmetros no tempo de execução da declaração.  
  
 Para obter mais informações, consulte [SQLParamData](../../odbc/reference/syntax/sqlparamdata-function.md) no *Programador ODBC*.
