# <a name="group-by-expression-list"></a>Lista de expressão GROUP BY
GROUP BY dá suporte a uma lista de expressões, bem como a um nome de coluna. A lista de seleção de uma instrução SELECT que tem uma cláusula GROUP BY só pode incluir expressões da cláusula GROUP BY ou funções set.
