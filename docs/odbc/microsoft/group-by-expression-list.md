# <a name="group-by-expression-list"></a>Lista de expressão GROUP BY
GROUP BY suporta uma lista de expressão, bem como um nome de coluna. A lista selecionada de uma declaração SELECT que tenha uma cláusula GROUP BY só pode incluir expressões da cláusula GROUP BY ou funções definidas.
