# <a name="sorting-limitations"></a>Limitações de classificação
O comprimento máximo de uma chave de tipo em uma cláusula DE GRUPO POR, ORDEM POR cláusula, declaração SELECT DISTINCT ou adesão externa é de 255 bytes; o comprimento máximo de todas as teclas de tipo em uma linha de classificação é de 65.500 bytes.
