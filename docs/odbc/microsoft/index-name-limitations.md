# <a name="index-name-limitations"></a>Limitações do nome do índice
Quando o driver Paradoxé é usado, um índice primário deve ter o mesmo nome da tabela sobre a qual é definido. Outros índices únicos ou não únicos devem ter o mesmo nome da tabela sobre a qual são definidos.
