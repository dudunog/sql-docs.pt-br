# <a name="scalar-functions"></a>Funções escalares
Os drivers de banco de dados de desktop do Microsoft ODBC suportam as seguintes funções escalares:  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 Para obter informações sobre os argumentos e valores de retorno das funções escalares, consulte o apêndice F da *Referência do Programador ODBC.*
