# <a name="sqlnumparams-visual-foxpro-odbc-driver"></a>SQLNumParams (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado em [referência da API do ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: completo  
  
 Conformidade da API ODBC: nível 2  
  
 Retorna o número de parâmetros em uma instrução SQL. O número de parâmetros deve ser igual ao número de pontos de interrogação na instrução SQL passado para [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 Para obter mais informações sobre a gramática SQL, consulte [suporte à gramática SQL ODBC](../../odbc/microsoft/supported-odbc-sql-grammar-visual-foxpro-odbc-driver.md). Para obter mais informações sobre essa função, consulte [SQLNumParams](../../odbc/reference/syntax/sqlnumparams-function.md) na *referência do programador de ODBC*.
