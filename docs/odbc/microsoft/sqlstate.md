# <a name="sqlstate"></a>SQLSTATE
Para erros que ocorrem na fonte de dados, o driver ODBC mapeia o erro nativo retornado para o SQLSTATE apropriado. Para erros detectados pelo motorista ou pelo Driver Manager, o driver ou driver manager da ODBC gera o SQLSTATE apropriado.
