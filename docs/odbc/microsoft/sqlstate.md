# <a name="sqlstate"></a>SQLSTATE
Para erros que ocorrem na fonte de dados, o driver ODBC mapeia o erro nativo retornado para o SQLSTATE apropriado. Para erros detectados pelo driver ou pelo Gerenciador de driver, o driver ODBC ou o Gerenciador de driver gera o SQLSTATE apropriado.
