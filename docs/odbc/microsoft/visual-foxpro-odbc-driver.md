# <a name="visual-foxpro-odbc-driver"></a>Driver ODBC do Visual FoxPro
O Microsoft Visual FoxPro é um ambiente poderoso orientado a objetos para construção de bancos de dados e desenvolvimento de aplicativos. O Microsoft Visual FoxPro ODBC Driver permite que os aplicativos abram, consultam e atualizem dados no Visual FoxPro e versões anteriores do FoxPro através da interface Open Database Connectivity (ODBC).  
  
 Por exemplo, com o Microsoft Visual FoxPro ODBC Driver você pode fazer o seguinte:  
  
-   Use o Microsoft Query para consultar e atualizar os dados do Visual FoxPro das planilhas do Microsoft Excel.  
  
-   Crie cartas de mesclagem de e-mail usando dados do Visual FoxPro com o Microsoft Word.  
  
-   Consulte e atualize as visualizações e tabelas do Visual FoxPro do Microsoft Access.  
  
-   Use o Visual FoxPro como o armazenamento de dados para os aplicativos Microsoft Visual Basic, Microsoft Visual C++e C.
