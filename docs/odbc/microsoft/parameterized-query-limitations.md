# <a name="parameterized-query-limitations"></a>Limitações da consulta parametrizada
Quando o driver do Microsoft Access é usado, uma consulta parametrizada pode ser chamada usando a seguinte sintaxe: chamar *Query-Name* [(*parâmetro*[,*Parameter*]...)].
