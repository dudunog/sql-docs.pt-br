# <a name="parameterized-query-limitations"></a>Limitações da consulta parametrizada
Quando o driver microsoft access é usado, uma consulta parametrizada pode ser chamada usando a seguinte sintaxe: *CALL query-name* [(parâmetro [,*parâmetro*]...].*parameter*
