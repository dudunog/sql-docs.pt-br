# <a name="files-to-use-with-the-desktop-database-drivers"></a>Arquivos a usar com os drivers de banco de dados de área de trabalho
Os Drivers de banco de dados desktop do ODBC são usados com o Microsoft Jet Database Engine, os arquivos ISAM instalados e os arquivos Microsoft ODBC. Nem todos são necessários para a redistribuição do produto, e nem todos são componentes principais.  
  
|Para obter informações sobre|Consulte|  
|---------------------------|---------|  
|Arquivos usados por cada driver de banco de dados da área de trabalho|Arquivos de driver ODBC|  
|Arquivos microsoft jet usados por cada driver de banco de dados de desktop|Arquivos microsoft jet|  
|Arquivos de driver IISAM que fornecem acesso a dados entre o Microsoft Jet e os outros DBMS|Arquivos de driver ISAM instalados|
