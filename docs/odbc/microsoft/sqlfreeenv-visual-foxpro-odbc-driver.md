# <a name="sqlfreeenv-visual-foxpro-odbc-driver"></a>SQLFreeEnv (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível do núcleo  
  
 Fecha o Visual FoxPro ODBC Driver e libera toda a memória associada ao driver.  
  
 Para obter mais informações, consulte [SQLFreeEnv](../../odbc/reference/syntax/sqlfreeenv-function.md) no *Programador ODBC .*
