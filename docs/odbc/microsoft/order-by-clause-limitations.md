# <a name="order-by-clause-limitations"></a>Limitações da cláusula ORDER BY
Se uma declaração SELECT contiver uma cláusula GROUP BY e uma cláusula ORDER BY, a cláusula ORDER BY poderá conter apenas uma coluna no conjunto de resultados ou uma expressão na cláusula GROUP BY.
