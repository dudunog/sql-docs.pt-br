# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível 1  
  
 Associa um buffer com um marcador de parâmetro em uma declaração SQL. O driver Visual FoxPro ODBC suporta parâmetros de entrada conforme especificado pelo argumento *fParamType.*  
  
 Para obter mais informações, consulte [SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) no *Programador ODBC .*
