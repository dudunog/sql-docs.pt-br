# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado em [referência da API do ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: completo  
  
 Conformidade da API ODBC: nível 1  
  
 Associa um buffer a um marcador de parâmetro em uma instrução SQL. O driver ODBC do Visual FoxPro dá suporte a parâmetros de entrada, conforme especificado pelo argumento *fParamType* .  
  
 Para obter mais informações, consulte [SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) na *referência do programador de ODBC*.
