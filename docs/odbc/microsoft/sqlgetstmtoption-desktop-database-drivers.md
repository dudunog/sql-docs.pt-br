# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (Drivers de banco de dados de área de trabalho)
Os marcadores retornados por uma *opção fOpção* de SQL_GETBOOKMARK só são válidos enquanto a consulta estiver aberta e são invalidados quando a consulta é reemitida. Marcadores persistentes não são suportados.
