# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver Visual FoxPro ODBC. Para obter informações gerais sobre esta função, consulte o tópico apropriado em [Referência à API oDBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: Completo  
  
 Conformidade da API ODBC: Nível do núcleo  
  
 Recupera uma linha de um resultado definido nos locais especificados pelas chamadas anteriores para [SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md). Prepara o driver para uma chamada para [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) para as colunas não vinculadas.  
  
 Para obter mais informações, consulte [SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) no *Programador ODBC*.
