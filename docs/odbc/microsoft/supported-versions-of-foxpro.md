# <a name="supported-versions-of-foxpro"></a>Versões do FoxPro com suporte
Você pode usar o Driver ODBC® ® Microsoft ® para acessar dados armazenados em tabelas FoxPro. As seguintes versões de dados FoxPro são suportadas:  
  
-   2,0  
  
-   2.5  
  
-   2.6  
  
-   Visual FoxPro (todas as versões)  
  
 Quando você acessa dados armazenados no Visual FoxPro, você pode optar por se conectar a um [banco de dados](../../odbc/microsoft/visual-foxpro-terminology.md) que contenha zero ou mais [tabelas](../../odbc/microsoft/visual-foxpro-terminology.md) ou a um diretório de [tabelas gratuitas](../../odbc/microsoft/visual-foxpro-terminology.md).  
  
 Para obter mais informações sobre como se conectar a uma fonte de dados, consulte [Adicionando uma fonte de dados Visual FoxPro](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md).  
  
 Para baixar o driver, consulte [o site de download visual FoxPro ODBC Driver](https://go.microsoft.com/fwlink/?LinkId=121318).
