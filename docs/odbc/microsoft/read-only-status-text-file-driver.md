# <a name="read-only-status-text-file-driver"></a>Status somente leitura (Driver de Arquivo de texto)
Quando o driver texto é usado, as tabelas de origem de dados são somente leitura. No entanto, os aplicativos podem executar inserções para tabelas de texto.
