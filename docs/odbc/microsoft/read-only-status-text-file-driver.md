# <a name="read-only-status-text-file-driver"></a>Status somente leitura (Driver de Arquivo de texto)
Quando o driver de texto é usado, as tabelas de fontes de dados são somente leitura. No entanto, os aplicativos podem executar inserções para tabelas de texto.
