# <a name="sqlspecialcolumns-desktop-database-drivers"></a>SQLSpecialColumns (Drivers de banco de dados de área de trabalho)
Um índice único será devolvido (se existir) para a bandeira SQL_BEST_ROWID em *fColType*. Nenhum conjunto de resultados será devolvido para a bandeira SQL_ROWVER.  
  
 Todas as linhas de id têm um escopo de SQL_SCOPE_CURROW.  
  
 A correspondência de padrões não é suportada para o argumento *szTableQualifier* ou *szTableName.*
