# <a name="viewing-drivers"></a>Exibir drivers
Para configurar fontes de dados, você deve ter instalado pelo menos um driver em seu sistema. A guia **Drivers** na caixa de diálogo **Administrador de Origem de Dados do ODBC** lista todos os drivers instalados no computador, incluindo o nome, a versão, a empresa, o nome do arquivo e a data de criação do arquivo de cada driver. Você deve usar o programa de configuração do driver para adicionar ou excluir um driver do seu sistema. Para obter mais informações sobre a modificação de drivers, consulte [Gerenciando fontes de dados](../../odbc/admin/managing-data-sources.md).
