# <a name="cstring-class"></a>Classe CString
Como objetos da classe **CString** na Microsoft® Visual C++® são assinados e os argumentos de seqüência em funções ODBC não são assinados, aplicativos que passam objetos **CString** para funções ODBC sem lançá-los receberão avisos do compilador.
