# <a name="sql-statements"></a>Instruções SQL
Os aplicativos ODBC executam quase todo o acesso ao banco de dados executando instruções SQL. A forma dessas declarações - codificadas ou construídas em tempo de execução, interoperáveis ou específicas da fonte de dados, e assim por diante - depende das necessidades do aplicativo.
