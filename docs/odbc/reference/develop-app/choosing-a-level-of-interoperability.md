# <a name="choosing-a-level-of-interoperability"></a>Escolher um nível de interoperabilidade
Supondo que o aplicativo use ODBC, o próximo passo é determinar qual nível de interoperabilidade é necessário. O nível básico de interoperabilidade é geralmente uma função do tipo de aplicação: aplicações personalizadas tendem a não ser interoperáveis, aplicações verticais tendem a ser interoperáveis entre um número limitado de DBMSs, e aplicações genéricas tendem a ser interoperáveis entre todos os DBMSs.  
  
 Esta seção contém os seguintes tópicos.

-   [Aplicativos verticais](../../../odbc/reference/develop-app/vertical-applications.md)
