# <a name="tables-and-views"></a>Tabelas e Exibições
Nas funções ODBC, tabelas e visualizações são intercambiáveis. A *tabela* de termos é usada tanto para tabelas quanto para visualizações, exceto quando o termo *exibição* é usado explicitamente.
