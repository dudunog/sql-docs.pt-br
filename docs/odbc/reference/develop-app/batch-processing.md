# <a name="batch-processing"></a>Processamento em lote
O suporte para lotes de instruções SQL não é generalizado, portanto, aplicativos interoperáveis devem usá-los condicionalmente ou não. Para obter mais informações, consulte [executando lotes](../../../odbc/reference/develop-app/executing-batches.md).
