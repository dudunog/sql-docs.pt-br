# <a name="backward-compatibility-and-standards-compliance"></a>Compatibilidade com versões anteriores e a conformidade com padrões
Compatibilidade retrógrada é a capacidade dos componentes ODBC mais novos de trabalhar com componentes ODBC antigos. As seções a seguir discutem como esses componentes são afetados pelas alterações no ODBC *3.x*. As informações contidas neles abordam principalmente a escrita de um aplicativo ODBC *3.x* e como os problemas de compatibilidade retroativa são tratados pelos drivers ODBC. Para obter orientações específicas sobre como problemas de compatibilidade retrógrada afetam a escrita de um driver ODBC *3.x,* consulte [Apêndice G: Diretrizes do driver para compatibilidade retrógrada](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md).
