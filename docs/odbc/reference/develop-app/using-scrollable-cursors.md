# <a name="using-scrollable-cursors"></a>Usar cursores roláveis
O uso de um cursor rolável requer estas três etapas:  
  
1.  Determine os recursos do cursor.  
  
2.  Configure o cursor.  
  
3.  Rolar e buscar linhas.
