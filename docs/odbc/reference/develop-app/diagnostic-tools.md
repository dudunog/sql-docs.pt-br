# <a name="diagnostic-tools"></a>Ferramentas de Diagnóstico
Duas instalações podem auxiliar no diagnóstico de problemas em aplicações ODBC. Ambos estão disponíveis no Administrador ODBC e implementados pelo Driver Manager. *O rastreamento* fornece um meio de gravar em um arquivo de log a seqüência de chamadas de função. Visual Studio Analyzer permite a análise de informações sobre a interação de componentes em um ambiente distribuído.  
  
 Esta seção contém os seguintes tópicos.  
  
-   [Rastreamento](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Analisador do Visual Studio](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
