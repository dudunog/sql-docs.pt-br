# <a name="conformance-levels"></a>Níveis de conformidade
Os drivers ODBC dão ao aplicativo acesso a diversas fontes de dados. Cada driver permite que o aplicativo determine no tempo de execução quais os recursos do ODBC e o que a gramática SQL que o driver e cada fonte de dados suportam. Este não é um requisito de aplicativos projetados para trabalhar com um único driver ou um pequeno conjunto conhecido de drivers, porque esses aplicativos podem simplesmente ser escritos para as capacidades desse motorista ou drivers. Para ajudar os aplicativos a descobrir os recursos de driver e fonte de dados, duas áreas de conformidade estão disponíveis: a interface ODBC e a gramática SQL.
