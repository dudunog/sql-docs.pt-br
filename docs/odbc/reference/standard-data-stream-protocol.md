# <a name="standard-data-stream-protocol"></a>Protocolo de fluxo de dados padrão
Um protocolo padrão de fluxo de dados é uma maneira de acessar dados em DBMSs heterogêneos. Na verdade, já existe um protocolo padrão de fluxo de dados:

- O padrão ANSI/ISO Remote Database Access (RDA): [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Embora o sistema ANSI/ISO mostre-se promissor, ele não é amplamente implementado hoje.
