# <a name="translation-dll-api-reference"></a>Referência de API de DLL de conversão
Esta seção descreve a sintaxe da API DLL de tradução, que consiste em duas funções: **SQLDriverToDataSource** e **SQLDataSourceToDriver**. Essas funções devem ser incluídas na DLL que executa a tradução para o driver.  
  
 Esta seção contém os seguintes tópicos.

-   [Função SQLDriverToDataSource](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)
