# <a name="sqlcolattributes-function"></a>Função SQLColAttributes
**Conformidade**  
 Versão introduzida: ODBC 1.0 Normas Conformidade: Preterido  
  
 **Resumo**  
 Em ODBC 3. *x*, a função ODBC 2.0 **SQLColAttributes** foi substituída por **SQLColAttribute**. Para obter mais informações, consulte [SQLColAttribute Function .](../../../odbc/reference/syntax/sqlcolattribute-function.md)  
  
> [!NOTE]  
>  Para obter mais informações sobre o que o Driver Manager mapeia essa função para quando um ODBC 2. *x* aplicativo está trabalhando com um ODBC 3. *x* driver, consulte [Mapeamento de Funções Depreciadas](../../../odbc/reference/appendixes/mapping-deprecated-functions.md) no Apêndice G: Diretrizes do driver para compatibilidade retrógrada.  
  
 Consulte [As informações de 64 bits do ODBC,](../../../odbc/reference/odbc-64-bit-information.md)se o aplicativo for executado em um sistema operacional de 64 bits.
