# <a name="literals-in-odbc"></a>Literais no ODBC
A sintaxe nas seções a seguir é usada para literais de intervalo e numéricos em ODBC. Esta sintaxe é fornecida aqui como um auxílio aos escritores de driver quando as conversões são realizadas de um tipo de string de caractere para um tipo numérico ou de intervalo, ou de um tipo numérico ou de intervalo para um tipo de string de caractere. Para obter mais informações, consulte [Interval Literals](../../../odbc/reference/appendixes/interval-literals.md) e [Numeric Literals](../../../odbc/reference/appendixes/numeric-literals.md) no apêndice D: Tipos de dados.  
  
 Este apêndice contém os seguintes tópicos.  
  
-   [Sintaxe literal de intervalo](../../../odbc/reference/appendixes/interval-literal-syntax.md)  
  
-   [Sintaxe literal numérica](../../../odbc/reference/appendixes/numeric-literal-syntax.md)
