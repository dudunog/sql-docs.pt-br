# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations e a Biblioteca de cursores
> [!IMPORTANT]  
>  Esse recurso será removido em uma versão futura do Windows. Evite usar esse recurso em novos trabalhos de desenvolvimento e planeje modificar aplicativos que atualmente usam esse recurso. A Microsoft recomenda o uso da funcionalidade do cursor do driver.  
  
 A biblioteca do cursor não suporta chamar **SQLBulkOperations**. Para obter informações gerais sobre **sqlbulkoperations,** consulte [SQLBulkOperations Function](../../../odbc/reference/syntax/sqlbulkoperations-function.md).
