# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations e a Biblioteca de cursores
> [!IMPORTANT]  
>  Este recurso será removido em uma versão futura do Windows. Evite usar esse recurso em novos trabalhos de desenvolvimento e planeje modificar os aplicativos que atualmente usam esse recurso. A Microsoft recomenda usar a funcionalidade de cursor do driver.  
  
 A biblioteca de cursores não dá suporte à chamada de **SQLBulkOperations**. Para obter informações gerais sobre **SQLBulkOperations**, consulte [SQLBulkOperations function](../../../odbc/reference/syntax/sqlbulkoperations-function.md).
