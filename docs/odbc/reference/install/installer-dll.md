# <a name="installer-dll"></a>DLL do Instalador
> [!NOTE]  
>  Começando com o Windows XP e o Windows Server 2003, o ODBC está incluído no sistema operacional windows. Você só deve instalar explicitamente o ODBC em versões anteriores do Windows.  
  
 O instalador DLL contém funções para instalar e remover componentes ODBC, manter informações cadastrais sobre esses componentes e manter informações cadastrais sobre fontes de dados. Ele é escrito pela Microsoft e pode ser redistribuído. Para obter uma descrição completa das funções no instalador DLL, consulte [Installer DLL API Reference](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
