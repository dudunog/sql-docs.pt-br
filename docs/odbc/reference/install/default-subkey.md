# <a name="default-subkey"></a>Subchave padrão
O registro pode especificar uma fonte de dados padrão com a subchave padrão. Essa subchave é um caso especial de uma subchave de especificação de fonte de dados e tem os mesmos valores de qualquer outra subchave de especificação de fonte de dados. A única diferença é que ela não está listada como um valor na subchave fontes de dados ODBC.
