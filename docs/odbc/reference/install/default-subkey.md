# <a name="default-subkey"></a>Subchave padrão
O registro pode especificar uma fonte de dados padrão com a subchave Padrão. Esta subchave é um caso especial de uma subchave de especificação de origem de dados e tem os mesmos valores que qualquer outra subchave de especificação de origem de dados. A única diferença é que ele não está listado como um valor sob a sub-chave ODBC Data Sources.
