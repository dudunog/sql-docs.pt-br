# <a name="server-related-dynamic-management-views-and-functions-transact-sql"></a>Funções e exibições de gerenciamento dinâmico relacionadas ao servidor (Transact-SQL)
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  Esta seção contém as exibições de gerenciamento dinâmico que estão associadas aos serviços SQL Server, Texto completo e  SQL Server Agent que estão instalados no servidor host. Você pode usar essas exibições para retornar informações de propriedade para esses serviços. Essas exibições também contêm informações sobre configuração, instalação e arquivo de despejo de memória.
