# <a name="localdb_error_invalid_instance_name"></a>LOCALDB_ERROR_INVALID_INSTANCE_NAME
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>Detalhes  
  
| Atributo | Valor |
| --------- | ----- |
|Nome do Produto|SQL Server|  
|ID do evento|283|  
|Origem do Evento|Runtime de banco de dados local do SQL Server 12.0|  
|Componente|API do runtime de banco de dados local|  
|Texto da mensagem|O nome da instância contém caracteres inválidos ou é longo demais. Consulte a documentação da API.|  
  
## <a name="explanation"></a>Explicação  
 O nome da instância contém caracteres inválidos ou é longo demais.  
  
## <a name="user-action"></a>Ação do usuário  
 Consulte a documentação da API do runtime de banco de dados local em nomes legais para instâncias de runtime de banco de dados local.  Para obter informações adicionais, consulte [referência de API da instância LocalDB do SQL Server Express](../../relational-databases/express-localdb-instance-apis/sql-server-express-localdb-reference-instance-apis.md).
