# <a name="localdb_error_instance_busy"></a>LOCALDB_ERROR_INSTANCE_BUSY
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>Detalhes  
  
| Atributo | Valor |
| --------- | ----- |
|Nome do Produto|SQL Server|  
|ID do evento|274|  
|Origem do Evento|Runtime de banco de dados local do SQL Server 12.0|  
|Componente|API do runtime de banco de dados local|  
|Texto da mensagem|A operação solicitada na instância de Banco de Dados Local não pode ser executada porque a instância especificada está em uso no momento. Interrompa a instância e tente novamente.|  
  
## <a name="explanation"></a>Explicação  
 A instância especificada está em execução.  
  
## <a name="user-action"></a>Ação do usuário  
 Interrompa a instância especificada do Runtime do Banco de Dados Local e tente novamente.
