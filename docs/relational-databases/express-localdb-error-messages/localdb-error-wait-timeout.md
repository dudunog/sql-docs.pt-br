# <a name="localdb_error_wait_timeout"></a>LOCALDB_ERROR_WAIT_TIMEOUT
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>Detalhes  
  
| Atributo | Valor |
| --------- | ----- |
|Nome do Produto|SQL Server|  
|ID do evento|277|  
|Origem do Evento|Runtime de banco de dados local do SQL Server 12.0|  
|Componente|API do runtime de banco de dados local|  
|Texto da mensagem|Ocorreu um tempo limite dentro do método de API da instância do Banco de Dados Local.|  
  
## <a name="explanation"></a>Explicação  
 Tempo limite esgotado durante tentativa de obtenção de bloqueios de sincronização.  
  
## <a name="user-action"></a>Ação do usuário  
 Repita a operação solicitada novamente.
