# <a name="filestream-and-filetable-functions-transact-sql"></a>Funções Filestream e FileTable (Transact-SQL)
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  Esta seção descreve as funções do sistema relacionadas aos recursos FILESTREAM e FileTable.  
  
 [FileTableRootPath &#40;Transact-SQL&#41;](../../relational-databases/system-functions/filetablerootpath-transact-sql.md)  
 Retorna o caminho UNC no nível raiz de uma FileTable específica ou do banco de dados atual.  
  
 [GetFileNamespacePath &#40;Transact-SQL&#41;](../../relational-databases/system-functions/getfilenamespacepath-transact-sql.md)  
 Retorna o caminho UNC de um arquivo ou diretório em uma FileTable.  
  
 [GetPathLocator &#40;Transact-SQL&#41;](../../relational-databases/system-functions/getpathlocator-transact-sql.md)  
 Retorna a ID do localizador do caminho do arquivo ou diretório especificado em uma FileTable.  
  
 [PathName &#40;Transact-SQL&#41;](../../relational-databases/system-functions/pathname-transact-sql.md)  
 Retorna o caminho de um objeto binário grande FILESTREAM (BLOB).  
  
 Para obter mais informações sobre FileTables, veja [FileTables &#40;SQL Server&#41;](../../relational-databases/blob/filetables-sql-server.md).
