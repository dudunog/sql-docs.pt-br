# <a name="advancedproperties-property-sqlservice-class"></a>Propriedade AdvancedProperties (classe SqlService)
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  Obtém uma matriz de referências de objeto que contêm as propriedades avançadas para o objeto **SqlService** .  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object.AdvancedProperties [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Um objeto da [classe SqlService](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md) que representa o serviço.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Uma matriz de objetos de [classe SqlServiceAdvancedProperty](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) que contêm as propriedades avançadas para o objeto **SqlService** .
