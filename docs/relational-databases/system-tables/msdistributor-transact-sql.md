# <a name="msdistributor-transact-sql"></a>MSdistributor (Transact-SQL)
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  A tabela **MSdistributor** contém as propriedades do distribuidor. Essa tabela é armazenada no banco de dados **msdb** .  
  
|Nome da coluna|Tipo de dados|Descrição|  
|-----------------|---------------|-----------------|  
|**property**|**sysname**|O nome da propriedade|  
|**value**|**nvarchar (3000)**|O valor da propriedade|
