# <a name="mssqlserver_26013"></a>MSSQLSERVER_26013
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>Detalhes  
  
|||  
|-|-|  
|Nome do Produto|SQL Server|  
|ID do evento|26013|  
|Origem do Evento|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nome simbólico|SNI_SSL_READY|  
|Texto da mensagem|O certificado [Cert Hash(sha1) "%hs"] foi carregado com êxito para criptografia.|  
  
## <a name="explanation"></a>Explicação  
Essa é uma mensagem informativa. O certificado nomeado foi carregado com êxito no [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] e pode ser usado para criptografia.  
  
## <a name="user-action"></a>Ação do usuário  
Nenhuma ação é necessária.
