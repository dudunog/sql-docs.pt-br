# <a name="sysextended_procedures-transact-sql"></a>sys.extended_procedures (Transact-SQL)
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  Contém uma linha para cada objeto que é um procedimento armazenado estendido, com **Sys. Objects. Type** = X. Como os procedimentos armazenados estendidos são instalados no banco de dados **mestre** , eles ficam visíveis apenas desse contexto de banco de dados. Selecionar na exibição de **Sys.extended_procedures** em qualquer outro contexto de banco de dados retornará um conjunto de resultados vazio.  

  
|Nome da coluna|Tipo de dados|Descrição|  
|-----------------|---------------|-----------------|  
|**\<Columns inherited from sys.objects>**||Para obter uma lista de colunas que essa exibição herda, consulte [Sys. objects &#40;Transact-SQL&#41;](../../relational-databases/system-catalog-views/sys-objects-transact-sql.md).|  
|**dll_name**|**nvarchar(260)**|Nome da DLL, incluindo caminho, para este procedimento armazenado estendido.|
