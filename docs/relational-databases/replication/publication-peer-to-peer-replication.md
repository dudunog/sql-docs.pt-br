# <a name="publication-peer-to-peer-replication"></a>Publicação (replicação ponto a ponto)
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  A página **Publicação** exibe publicações transacionais habilitadas para replicação ponto a ponto. As publicações são habilitadas na página **Opções de Assinatura** da caixa de diálogo **Propriedades de Publicação** .  
  
## <a name="options"></a>Opções  
 **Publicador**  
 Exibe o servidor com o qual você está conectado. Para conectar a um servidor diferente, selecione **Encontrar Editor SQL Server**.  
  
 **Bancos de dados e publicações**  
 Exibe todos os bancos de dados no servidor que contêm pelo menos uma publicação habilitada para replicação ponto a ponto. Selecione uma publicação para continuar.
