# <a name="cursors-event-category"></a>Categoria de evento Cursors
[!INCLUDE [SQL Server - ASDB](../../includes/applies-to-version/sql-asdb.md)]
  A categoria de evento **Cursors** contém classes de evento usadas para monitorar o comportamento de cursores.  
  
## <a name="in-this-section"></a>Nesta seção  
  
|Tópico|Descrição|  
|-----------|-----------------|  
|[Classe de evento CursorClose](../../relational-databases/event-classes/cursorclose-event-class.md)|Descreve eventos de fechamento de cursor que ocorrem em cursores de interface de programação de aplicativo (API).|  
|[Classe de evento CursorExecute](../../relational-databases/event-classes/cursorexecute-event-class.md)|Descreve eventos de execução de cursor que ocorrem em cursores de API.|  
|[Classe de evento CursorImplicitConversion](../../relational-databases/event-classes/cursorimplicitconversion-event-class.md)|Descreve eventos de conversão implícita de cursor que ocorrem em cursores de API ou [!INCLUDE[tsql](../../includes/tsql-md.md)] .|  
|[Classe de evento CursorOpen](../../relational-databases/event-classes/cursoropen-event-class.md)|Descreve eventos de abertura de cursor que ocorrem em cursores de API.|  
|[Classe de evento CursorPrepare](../../relational-databases/event-classes/cursorprepare-event-class.md)|Descreve eventos de preparação de cursor que ocorrem em cursores de API.|  
|[Classe de evento CursorRecompile](../../relational-databases/event-classes/cursorrecompile-event-class.md)|Descreve eventos de recompilação de cursor que ocorrem em cursores de API.|  
|[Classe de evento CursorUnprepare](../../relational-databases/event-classes/cursorunprepare-event-class.md)|Descreve eventos de despreparação de cursor que ocorrem em cursores de API.|
