# <a name="create-queries-visual-database-tools"></a>Criar consultas (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
As consultas permitem que você recupere dados das tabelas e exibições em seu banco de dados. Você cria e trabalha com consultas no **Designer de Consulta e Exibição**, que é composto por quatro painéis: o [Painel de Diagrama](../../ssms/visual-db-tools/diagram-pane-visual-database-tools.md), o [Painel SQL](../../ssms/visual-db-tools/sql-pane-visual-database-tools.md), o [Painel de Critérios](../../ssms/visual-db-tools/criteria-pane-visual-database-tools.md)e o [Painel de Resultados](../../ssms/visual-db-tools/results-pane-visual-database-tools.md).  
  
### <a name="to-create-a-new-query"></a>Para criar uma consulta  
  
1.  No **Pesquisador de Objetos**, expanda o nó **Tabelas** do banco de dados que você deseja consultar. Clique com o botão direito do mouse na tabela que deseja consultar e clique em **Abrir Tabela**.  
  
2.  Para adicionar mais tabelas à consulta, no menu Designer de Consultas, selecione **Adicionar Tabela**.  
  
    > [!NOTE]  
    > Se você não conseguir ver os painéis **Diagrama**, **SQL**, **Critérios**ou **Resultados** , no menu Designer de Consultas, aponte para **Painel** e clique no painel que deseja abrir.  
  
3.  Na caixa de diálogo **Adicionar Tabela** , selecione as tabelas que você deseja consultar e clique em **Adicionar** para cada uma.  
  
4.  Após adicionar todas as tabelas que deseja consultar, clique em **Fechar**.  
  
    Para adicionar mais tabelas posteriormente, clique com o botão direito do mouse no espaço aberto do painel **Diagrama** e, no menu de atalho, clique em **Adicionar Tabela**.  
  
5.  No **Painel de Diagrama**, marque as caixas nos objetos com valor de tabela para cada coluna que você deseja consultar.  
  
6.  No menu Designer de Consultas, escolha **Executar SQL** para executar sua consulta.  
  
Para refinar ainda mais a sua consulta, você pode alterar o código SQL no **Painel SQL** ou escolher opções como ordem de classificação e aliases de coluna no **Painel de Critérios.**
