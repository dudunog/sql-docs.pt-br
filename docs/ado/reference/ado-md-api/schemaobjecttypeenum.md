# <a name="schemaobjecttypeenum"></a>SchemaObjectTypeEnum
Especifica o tipo de objeto a ser recuperado pelo método [GetSchemaObject](./getschemaobject-method-ado-md.md) .  
  
|Constante|Valor|Descrição|  
|--------------|-----------|-----------------|  
|**adObjectTypeDimension**|1|Um objeto [Dimension](./dimension-object-ado-md.md) .|  
|**adObjectTypeHierarchy**|2|Um objeto [Hierarchy](./hierarchy-object-ado-md.md) .|  
|**adObjectTypeLevel**|3|Um objeto de [nível](./level-object-ado-md.md) .|  
|**adObjectTypeMember**|4|Um objeto de [membro](./member-object-ado-md.md) .|
