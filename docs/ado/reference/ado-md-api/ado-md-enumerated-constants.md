# <a name="ado-md-enumerated-constants"></a>Constantes enumeradas do ADO MD
Para auxiliar na depuração, o ADO MD constantes enumeradas lista um valor para cada constante. No entanto, esse valor é puramente consultivo e pode ser alterado de uma versão do ADO MD para outra. Seu código só deve depender do nome, não do valor real, das constantes enumeradas.  
  
 As enumerações a seguir são definidas.  
  
-   [SchemaObjectTypeEnum](./schemaobjecttypeenum.md)  
  
-   [MemberTypeEnum](./membertypeenum.md)
