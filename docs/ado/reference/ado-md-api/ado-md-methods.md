# <a name="ado-md-methods"></a>Métodos do ADO MD

|Método|Descrição|  
|-|-|  
|[Fechar](./close-method-ado-md.md)|Fecha um células aberto.|  
|[GetSchemaObject](./getschemaobject-method-ado-md.md)|Recupera um objeto de esquema ADO MD (dimensão, hierarquia, nível ou membro) por seu nome exclusivo.|  
|[Abrir](./open-method-ado-md.md)|Recupera os resultados de uma consulta multidimensional e retorna os resultados para um células.|
