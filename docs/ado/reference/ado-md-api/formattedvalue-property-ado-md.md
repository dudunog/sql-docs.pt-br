# <a name="formattedvalue-property-ado-md"></a>Propriedade FormattedValue (ADO MD)
Indica a exibição formatada de um valor de [célula](../../../ado/reference/ado-md-api/cell-object-ado-md.md) .  
  
## <a name="return-values"></a>Valores de retorno  
 Retorna uma **cadeia de caracteres** e é somente leitura.  
  
## <a name="remarks"></a>Comentários  
 Use a propriedade **FormattedValue** para obter o valor de exibição formatado da propriedade [Value](../../../ado/reference/ado-md-api/value-property-ado-md.md) de um objeto [Cell](../../../ado/reference/ado-md-api/cell-object-ado-md.md) . Por exemplo, se o valor de uma célula fosse 1056,87, e esse valor representou um valor de dólar, **FormattedValue** seria $1056.87.
