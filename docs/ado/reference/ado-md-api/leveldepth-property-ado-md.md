# <a name="leveldepth-property-ado-md"></a>Propriedade LevelDepth (ADO MD)
Indica o número de níveis entre a raiz da hierarquia e um [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Valores de retorno  
 Retorna um inteiro **longo** e é somente leitura.  
  
## <a name="remarks"></a>Comentários  
 Use a propriedade **LevelDepth** para determinar a distância do objeto de [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md)a partir do nível raiz da hierarquia. O **LevelDepth**de um membro no nível raiz é 0. Isso corresponde à propriedade [Depth](../../../ado/reference/ado-md-api/depth-property-ado-md.md) de um objeto [Level](../../../ado/reference/ado-md-api/level-object-ado-md.md) .
