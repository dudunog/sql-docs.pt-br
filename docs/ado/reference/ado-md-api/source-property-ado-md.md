# <a name="source-property-ado-md"></a>Propriedade Source (ADO MD)
Indica a origem dos dados no [células](./cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define ou retorna uma **variante**e é leitura/gravação para objetos [células](./cellset-object-ado-md.md) fechados e somente leitura para objetos Open **células** . A **variante** deve conter uma **cadeia de caracteres**válida, por exemplo, uma consulta MDX.
