# <a name="compareenum"></a>CompareEnum
Especifica a posição relativa de dois registros representados por seus indicadores.  
  
|Constante|Valor|Descrição|  
|--------------|-----------|-----------------|  
|**adCompareEqual**|1|Indica que os indicadores são iguais.|  
|**adCompareGreaterThan**|2|Indica que o primeiro indicador é posterior ao segundo.|  
|**adCompareLessThan**|0|Indica que o primeiro indicador é anterior ao segundo.|  
|**adCompareNotComparable**|4|Indica que os indicadores não podem ser comparados.|  
|**adCompareNotEqual**|3|Indica que os indicadores não são iguais e não são ordenados.|
