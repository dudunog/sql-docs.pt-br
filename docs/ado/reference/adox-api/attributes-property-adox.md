# <a name="attributes-property-adox"></a>Propriedade Attributes (ADOX)
Descreve as características da coluna.  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define ou retorna um valor **longo** . O valor especifica as características da tabela que é representada pelo objeto de [coluna](../../../ado/reference/adox-api/column-object-adox.md) . O valor pode ser uma combinação de constantes [ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md) . O valor padrão é zero (**0**), que não é **adColFixed** nem **adColNullable**.  
  
## <a name="applies-to"></a>Aplica-se A  
  
- [Objeto Column (ADOX)](../../../ado/reference/adox-api/column-object-adox.md)
