# <a name="type-property-column-adox"></a>Propriedade Type (Column) (ADOX)
Indica o tipo de dados de uma coluna.  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define ou retorna um valor **longo** que pode ser uma das constantes [DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md) . O valor padrão é **adVarWChar**.  
  
## <a name="remarks"></a>Comentários  
 Essa propriedade é de leitura/gravação até que o objeto de [coluna](../../../ado/reference/adox-api/column-object-adox.md) seja anexado a uma coleção ou a outro objeto, após o qual ele é somente leitura.
