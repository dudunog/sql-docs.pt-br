# <a name="deleterule-property-adox"></a>Propriedade DeleteRule (ADOX)
Indica a ação executada quando uma chave primária é excluída.  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define e retorna um valor **longo** que pode ser uma das constantes [RuleEnum](./ruleenum.md) . O valor padrão é **adRINone**.  
  
## <a name="remarks"></a>Comentários  
 Esta propriedade é somente leitura em objetos de [chave](./key-object-adox.md) já anexados a uma coleção.  
  
## <a name="applies-to"></a>Aplica-se A  
 [Objeto Key (ADOX)](./key-object-adox.md)
