# <a name="operators---unary"></a>Operadores – unários
[!INCLUDE[ssas](../includes/applies-to-version/ssas.md)]

  Os operadores unários executam uma operação em um operando único, como retornar o valor negativo ou positivo de uma expressão numérica.  
  
 A tabela a seguir identifica os operadores unários aos quais a DMX oferece suporte.  
  
|Operador|Descrição|  
|--------------|-----------------|  
|[-&#40;&#41; &#40;MDX negativo&#41;](../mdx/negative-mdx.md)|Retorna o valor negativo de uma expressão numérica.|  
|[+ &#40;&#41; &#40;MDX positivo&#41;](../mdx/positive-mdx.md)|Retorna o valor positivo de uma expressão numérica.|
