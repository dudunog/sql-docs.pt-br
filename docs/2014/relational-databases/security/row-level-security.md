# <a name="row-level-security"></a>Segurança em nível de linha
  A Segurança em Nível de Linha permite aos clientes controlar o acesso às linhas em uma tabela de banco de dados com base nas características do usuário que executa uma consulta (por exemplo, uma associação de grupo ou um contexto de execução). A segurança em nível de linha está disponível agora no [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2016. Consulte [Segurança em nível de linha](https://msdn.microsoft.com/library/dn765131.aspx) na documentação atual para obter a descrição atual desse recurso.
