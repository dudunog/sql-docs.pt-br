# <a name="flagname-property-serversettingsgeneralflag-class"></a>Propriedade FlagName (classe ServerSettingsGeneralFlag)
  Obtém o nome do sinalizador.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.FlagName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Um objeto da [classe ServerSettingsGeneralFlag](serversettingsgeneralflag-class.md) que representa um sinalizador geral para as configurações de servidor.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor da cadeia de caracteres que especifica o nome do sinalizador.
