# <a name="setdisable-method-servernetworkprotocolipaddress-class"></a>Método SetDisable (classe ServerNetworkProtocolIPAddress)
  Desabilita o endereço IP.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.SetDisable()  
  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Um objeto [ServerNetworkProtocolIPAdress Class] ServerNetworkProtocolIPAddress-class.md) que representa um endereço IP para o protocolo de rede em uma instância do [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor uint32, que é 0 se o serviço tiver sido modificado com êxito, 1 se a solicitação não tiver suporte e qualquer outro número para indicar um erro.
