# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  A interface IBCPSession2 é uma extensão de IBCPSession que fornece uma função de membro que é uma alternativa à chamada de IBCPSession::BCPColFmt para cada coluna.  IBCPSession2 herda de IBCPSession e adiciona um novo método: [IBCPSession2:: BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md).
