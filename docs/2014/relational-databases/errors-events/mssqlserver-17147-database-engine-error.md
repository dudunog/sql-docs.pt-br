# <a name="mssqlserver_17147"></a>MSSQLSERVER_17147
    
## <a name="details"></a>Detalhes  
  
|Atributo|Valor|  
|-|-|  
|Nome do Produto|SQL Server|  
|ID do evento|17147|  
|Origem do Evento|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nome simbólico|STARTUP_SERVERKILLED|  
|Texto da mensagem|O SQL Server está sendo encerrado devido a um desligamento do sistema. Essa mensagem é apenas informativa. Não é necessária nenhuma ação do usuário.|
