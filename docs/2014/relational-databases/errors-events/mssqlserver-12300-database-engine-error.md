# <a name="mssqlserver_12300"></a>MSSQLSERVER_12300
    
## <a name="details"></a>Detalhes  
  
|||  
|-|-|  
|Nome do Produto|SQL Server|  
|ID do evento|12300|  
|Origem do Evento|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nome simbólico|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|Texto da mensagem|Não há suporte para colunas computadas com '*construct*'.|  
  
## <a name="user-action"></a>Ação do usuário  
 As colunas computadas não têm suporte com tabelas com otimização de memória.
