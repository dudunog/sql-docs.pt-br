# <a name="isinitialized-property-wmi-msreportserver_configurationsetting"></a>Propriedade IsInitialized (WMI MSReportServer_ConfigurationSetting)
  Indica se a instância do servidor de relatório foi inicializada. Somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim IsInitialized As Boolean  
```  
  
```csharp  
public Boolean DatabaseLogonName;  
```  
  
## <a name="property-values"></a>Valores da propriedade  
 Um objeto `Boolean` que indica se a instância do servidor de relatório será inicializada.
