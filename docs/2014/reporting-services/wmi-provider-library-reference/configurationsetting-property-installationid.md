# <a name="installationid-property-wmi-msreportserver_configurationsetting"></a>Propriedade InstallationID (WMI MSReportServer_ConfigurationSetting)
  Esta propriedade retorna um identificador exclusivo de uma instância de servidor de relatório específica.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
