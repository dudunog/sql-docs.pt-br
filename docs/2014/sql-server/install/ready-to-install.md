# <a name="ready-to-install"></a>Pronto para instalar
  A página Pronto para Instalar apresenta um resumo dos recursos a serem instalados ou alterados pela instalação do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Opções  
 Você tem as seguintes opções nessa página:  
  
-   Para continuar a Instalação, clique em **Instalar**.  
  
-   Para alterar suas configurações de instalação, clique em **Voltar**.  
  
-   Para sair da instalação, clique em **Cancelar**.
