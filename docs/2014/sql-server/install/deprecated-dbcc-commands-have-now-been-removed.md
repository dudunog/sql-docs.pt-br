# <a name="deprecated-dbcc-commands-have-now-been-removed"></a>Comandos DBCC anteriormente substituídos foram removidos
  A tabela a seguir lista os comandos DBCC que foram anunciados como substituídos no [!INCLUDE[ssVersion2000](../../includes/ssversion2000-md.md)]. Esses comandos foram removidos do [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] e de versões posteriores. Você deve atualizar aplicativos e scripts para remover referências a esses comandos substituídos.  
  
||  
|-|  
|DBCC NEWALLOC|  
|DBCC ROWLOCK|  
|DBCC TEXTALLOC|  
|DBCC TEXTALL|
