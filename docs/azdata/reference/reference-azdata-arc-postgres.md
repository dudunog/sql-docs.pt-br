O artigo a seguir fornece referência para os comandos **sql** na ferramenta **azdata**. Para obter mais informações sobre outros comandos de **azdata**, confira [referência de azdata](reference-azdata.md)

## <a name="commands"></a>Comandos

|Comando|Descrição|
| --- | --- |
[azdata arc postgres server](reference-azdata-arc-postgres-server.md) | Gerenciar grupos de servidores PostgreSQL.
[azdata arc postgres backup](reference-azdata-arc-postgres-backup.md) | Gerenciar backups do grupo de servidores PostgreSQL.
[azdata arc postgres endpoint](reference-azdata-arc-postgres-endpoint.md) | Gerenciar pontos de extremidade de grupos de servidores PostgreSQL.
