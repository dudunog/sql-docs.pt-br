# <a name="azdata-arc"></a>azdata arc

Aplica-se ao `azdata`

O artigo a seguir fornece referência para os comandos **sql** na ferramenta **azdata**. Para obter mais informações sobre outros comandos de **azdata**, confira [referência de azdata](reference-azdata.md)

## <a name="commands"></a>Comandos

|Comando|Descrição|
| --- | --- |
[azdata arc sql](reference-azdata-arc-sql.md) | Criar, excluir e gerenciar recursos de SQL.
[azdata arc dc](reference-azdata-arc-dc.md) | Criar, excluir e gerenciar controladores de dados.
[azdata arc resource-kind](reference-azdata-arc-resource-kind.md) | Comandos resource-kind para definir e modelar recursos personalizados no cluster.
[azdata arc resource](reference-azdata-arc-resource.md) | Comandos de recurso para criar e gerenciar recursos personalizados no cluster.
[azdata arc postgres](reference-azdata-arc-postgres.md) | Criar, excluir e gerenciar grupos de servidores PostgreSQL.
