# <a name="assignment-operators"></a>Operadores de atribuição


  Na linguagem MDX, o operador de atribuição é o sinal de igual (=).  
  
 O operador de atribuição é usado para atribuir valores a subcubos em script MDX. Para obter mais informações, consulte [Conceitos básicos do script MDX &#40;Analysis Services&#41;](/analysis-services/multidimensional-models/mdx/mdx-scripting-fundamentals-analysis-services).
