# <a name="msreportserver_instance-properties---editionid"></a>Propriedades de MSReportServer_Instance – EditionID
  Retorna a edição do servidor de relatório. Somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>Valor da propriedade  
 Uma **uint** que contém a edição do servidor de relatório.
