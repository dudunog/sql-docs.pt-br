# <a name="overview-of-the-jdbc-driver"></a>Visão geral do JDBC Driver

[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

O Microsoft JDBC Driver para SQL Server é um driver compatível com o JDBC (Java Database Connectivity) 4.2 Tipo 4 que fornece acesso a dados robusto aos bancos de dados do SQL Server e SQL do Azure.  

## <a name="redistribution"></a>Redistribuição

As versões 6.0 e posteriores do JDBC Driver são redistribuíveis. Confira a cláusula sobre _Código Distribuível_ nos contratos de licença.

As versões 4.x do JDBC Driver são obsoletas. O suporte para as versões 4.x expirou antes de 2018.
