# <a name="position-method-sqlservernclob"></a>Método position (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera a posição do caractere do objeto **NClob** ou da subcadeia de caracteres especificada no **NClob** com base na posição inicial.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|DESCRIÇÃO|  
|----------|-----------------|  
|[Método position &#40;java.sql.NClob, long&#41;](../../../connect/jdbc/reference/position-method-java-sql-nclob-long.md)|Recupera a posição do caractere no qual o objeto **NClob** especificado *searchstr* aparece neste objeto **NClob**.|  
|[Método position &#40;java.lang.String, long&#41; &#40;SQLServerNClob&#41;](../../../connect/jdbc/reference/position-method-java-lang-string-long-sqlservernclob.md)|Recupera a posição de caractere na qual a substring *searchstr* especificada aparece no valor **NCLOB** representado pelo objeto **NClob**.|
