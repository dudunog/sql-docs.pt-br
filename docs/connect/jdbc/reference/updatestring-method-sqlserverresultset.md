# <a name="updatestring-method-sqlserverresultset"></a>Método updateString (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Atualiza a coluna designada com um valor de **Cadeia de Caracteres**.

|Nome|Descrição|  
|----------|-----------------|  
|[updateString (int, java.lang.String)](../../../connect/jdbc/reference/updatestring-method-int-java-lang-string.md)|Atualiza a coluna designada com um valor de **Cadeia de Caracteres**, considerando o índice da coluna.|  
|[updateString (java.lang.String, java.lang.String)](../../../connect/jdbc/reference/updatestring-method-java-lang-string-java-lang-string.md)|Atualiza a coluna designada com um valor de **Cadeia de Caracteres**, considerando o nome da coluna.|
