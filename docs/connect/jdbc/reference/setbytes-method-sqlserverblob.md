# <a name="setbytes-method-sqlserverblob"></a>Método setBytes (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Grava a matriz de bytes fornecida no BLOB iniciando na posição determinada e, em seguida, retorna o número de bytes gravados.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|DESCRIÇÃO|  
|----------|-----------------|  
|[setBytes (long, byte&#91;&#93;)](../../../connect/jdbc/reference/setbytes-method-long-byte.md)|Grava a matriz de bytes fornecida no BLOB iniciando na posição determinada e, em seguida, retorna o número de bytes gravados.|  
|[setBytes (long, byte&#91;&#93;, int, int)](../../../connect/jdbc/reference/setbytes-method-long-byte-int-int.md)|Grava toda ou parte da matriz de bytes fornecida no BLOB iniciando na posição, no deslocamento e no comprimento fornecidos e, em seguida, retorna o número de bytes gravados.|
