# <a name="free-method-sqlservernclob"></a>Método free (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Esse método libera o objeto NCLOB e os recursos que ele contém.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public void free()  
```  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentários  
 Esse método gratuito é especificado pelo método gratuito na interface java.sql.NClob.
