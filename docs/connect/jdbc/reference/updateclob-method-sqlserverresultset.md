# <a name="updateclob-method-sqlserverresultset"></a>Método updateClob (SQLServerResultSet)

|[updateClob (java.lang.String, java.sql.Clob)](../../../connect/jdbc/reference/updateclob-method-java-lang-string-java-sql-clob.md)|Atualiza a coluna designada com um valor java.sql.Clob considerando o nome da coluna.|
